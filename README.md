# HeyChat

- [x] SignIn Screen
- [x] SignUp Screen
- [ ] ChatRoom Screen
- [ ] Messages Screen
- [ ] Search Screen

<br>

1. Clone o repositório

```bash
git clone https://github.com/FixRuan/HeyChat
```

2. Acesse a pasta do projeto

```bash
cd HeyChat
```

3. Instale as dependências necessárias

```bash
yarn
```

4. Execute o projeto

```bash
yarn start
```
